Description: This is a basic client-server chat application that uses sockets to create a connection between the server and clients. It supports multiple clients within a local area network (LAN) or Wi-Fi. To use, the server-side code must be running, and the client-side code should connect using the server's IP address. Created by mng48301.

Features:

  • Supports multiple clients.
  • Real-time messaging between clients.
  • Dynamic client index updates.

Code Snippet: Client-Side Implementation


import socket
import tkinter as tk
import threading

# Connect to server
client_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
server_ip_address = ""  # Replace with server IP
port_address = 12345
client_socket.connect((server_ip_address, port_address))

# Handle incoming messages
def handle_messages():
    while True:
        incoming_message = client_socket.recv(2000).decode()
        print("Message received:", incoming_message)

threading.Thread(target=handle_messages).start()
