# Metric Convertor #

Simple command-line convertor that transform metric units into imperial. 

 ### Supports ###
- kg to lb (kg/lb)
- gram to ounces(g/oz)
- km to mile(km/mi)
- mm to inch(mm/in)

at any time, type 'exit' to quit"

### SETUP ### 

Run ```java -jar yourfile.jar``` in the terminal to execute file.
